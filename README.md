# vue-component-seed

Basis Vue reusable-component seed

[![vue2](https://img.shields.io/badge/vue-2.x-brightgreen.svg)](https://vuejs.org/)


```html
  <vue-component-seed msg="Example"></vue-component-seed>
```

### Install dependencies

```
$ npm install
```

### Building Your Element

```
$ npm run build
```

### Running Tests

```
$ npm run test
```
